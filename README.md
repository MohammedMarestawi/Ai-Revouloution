# AI-Revolution

A modern AI landing page built with **React**, **Vite**, and **Tailwind CSS**. This project demonstrates a clean UI, responsive layout, and reusable component structure.

## Live Demo
https://ai-revouloution-nu.vercel.app/

## Preview

A stylish AI website including:
- Hero section with call-to-action
- AI technology cards
- Benefits section
- Contact form
- Footer with resources

## Technologies Used

- React
- Vite
- Tailwind CSS
- ESLint
- Node.js

## Installation

Make sure you have **Node.js** installed.

npm install

## Run Development Server

npm run dev

## Build for Production

npm run build

## Project Structure

src/
 ├── components/
 ├── pages/
 └── App.jsx

## Features

- Responsive design
- Modern UI with gradients
- Reusable components
- Clean layout and spacing
- Simple and scalable structure
